## 概要

Go の実験コードを保存するレポジトリです

## ソフトウェアアーキテクチャ / ディレクトリ構成

- `internal/cmd/{command}/*.go`  
  - コマンドのエントリポイント (`main.go`) を格納します。  
  - 例:  
    - `internal/cmd/ex0001`: シグナル受信処理を `main` 関数で行うグレースフルシャットダウン実装。

## テストの実行方法

### ソースコード修正後は下記を実行してください

```bash
make fmt
make lint
make test
```
