# レポジトリのREADME.md

## 概要

実験コードを保存するレポジトリです

## ディレクトリ構成

- `apps/{app}`
  - 実験コードを保存するディレクトリ
- `apps/{app}/go`
  - Go のソースコードを保存するディレクトリ
  - go.mod,go.sum を保存してください

プレースホルダー

- app
  - 実験用のアプリケーション名

## ソースコード修正後の検証

ソースコード修正後は`apps/{app}`配下のディレクトリで、下記を実行してください

```bash
# ソースコードのスタイルを整形する
make fmt

# ソースコードを静的解析する
make lint

# ソースコードをテストする
make test
```
