# `apps/exp001/go`のREADME.md

## 概要

1つのmain.goだけで完結する断片的な実験コード集

## アーキテクチャ

- `internal/cmd/{command}/*.go`  
  - コマンドのエントリポイント (`main.go`) を格納します。  
  - 例:  
    - `internal/cmd/ex0001`: シグナル受信処理を `main` 関数で行うグレースフルシャットダウン実装。

## コマンド一覧

### `ex0001.cmd`

- ビルド: `make ex0001.cmd`
- 概要: シグナル受信とシャットダウン処理を `main` 関数で行うグレースフルシャットダウン実装。

### `ex0002.cmd`

- ビルド: `make ex0002.cmd`
- 概要: シグナル受信とシャットダウン処理を別ゴルーチンで行うグレースフルシャットダウン実装。
